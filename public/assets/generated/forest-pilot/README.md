# Forest pilot asset set

Original generated assets for the Three.js HD-2D forest pilot.

- `forest-dawn-far.webp`: distant battle background.
- `forest-mid.webp`: transparent tree and ruin billboard band.
- `forest-fore.webp`: transparent foreground vegetation frame.
- `world-map.webp`: campaign-map-specific panorama.

The PNG files are retained as editable source assets. Runtime files use WebP.

Art direction: clean hand-painted HD fantasy, rounded silhouettes, teal and
moss shadows, warm dawn rim light, low texture noise, no pixel-art treatment.
