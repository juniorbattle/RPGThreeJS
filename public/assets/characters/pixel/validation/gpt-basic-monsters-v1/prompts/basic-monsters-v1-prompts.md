# GPT external prompts — Basic monsters v1

Destination for generated raw images:

`public/assets/characters/pixel/validation/gpt-basic-monsters-v1/raw/`

Rules:

- Generate one image per prompt.
- Do not generate a spritesheet or atlas.
- Use a solid `#FF00FF` magenta background only.
- No scenery, no text, no UI, no ground plate, no cast shadow.
- The creature must be fully visible, centered, feet/paws/tail safely inside the canvas.
- Use a brighter readable tactical palette than the previous Serpent sprites.
- Avoid pure black except tiny accents.
- Add subtle warm rim light and clean highlights for readability against dark forest/burning village backgrounds.
- Style target: stylized pixel art fantasy tactical RPG, coherent with the current hero sprites.
- Canvas: 1086x1448 or 1024x1365.

## Shared negative prompt

Do not include: realistic rendering, painterly background, environment, text, labels, watermark, UI, multiple characters, multiple poses, sprite atlas, white background, transparent checkerboard, cropped body, cut off feet, extreme darkness, horror gore, boss-scale creature, undead, dragon, troll, demon, mythical legendary beast.
