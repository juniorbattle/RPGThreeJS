# Lot personnages clés — validation manuelle

Ce dossier isole les candidats de PNJ et de personnages narratifs hors runtime.

- `raw/` : sources GPT d'origine, conservées sans modification.
- `processed/` : aperçu transparent nettoyé par candidat.
- `boards/` : planches comparatives. Les identifiants `K01` à `K15` suivent l'ordre alphabétique des UUID de `raw/`.
- `rejected/` : réservé aux rejets validés manuellement.

Cette passe n'attribue pas d'identité narrative aux UUID ambigus et ne les promeut pas. Toute correspondance avec un personnage existant sera décidée après revue visuelle.

La provenance exacte des prompts n'a pas été reconstituée : aucun prompt n'est inventé dans cette passe.
